# 足球转播讨论页

一个轻量级的足球转播实时讨论页面，无需后端，纯前端实现。

## 功能

- 👤 **账号注册 / 登录** — 使用手机号或邮箱注册，接收（演示）验证码完成注册；支持密码登录
- 🔑 **忘记密码** — 通过手机号或邮箱验证码自助重置密码
- 🔒 **讨论区登录门控** — 需登录账号后才可发表评论，保障讨论质量
- 📺 **实时直播视频** — 内嵌视频播放器，支持多频道切换（生产环境替换为授权直播源）
- 📡 **正在直播** — 展示当前欧冠等赛事的实时比分与分钟数
- 📊 **实时数据** — 控球率、射门、角球等比赛数据对比
- 🔥 **热议话题** — 当前最热门的讨论话题榜单
- 💬 **讨论区** — 发表评论、点赞互动，支持 Ctrl+Enter 快捷发送
- 📢 **广告展示** — 顶部横幅广告、侧边栏广告卡片、底部多格广告条

## 账号系统说明

账号数据使用浏览器 `localStorage` 存储（纯前端演示）。密码通过 Web Crypto API（SHA-256）哈希后存储。  
验证码在演示模式下直接显示在页面上（生产环境应接入短信/邮件服务）。

## 使用

直接在浏览器中打开 `index.html` 即可，无需安装任何依赖。
